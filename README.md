# gestion_patrimoine
Application de gestion patrimoine, des inventaires
etape 1: tapez git clone https://github.com/Marcel1902/gestion_patrimoine.git dans votre terminal
etape 2: une fois terminée, ouvrir le dossier gestion_patrimoine dans Visual Studio code
etape 3: Ouvrir le terminal de Visual Studio code et tapez python manage.py runserver
etape: il y aura un lien "127.0.0.1:8000" dans le terminal, copier le et coller dans votre navigateur
